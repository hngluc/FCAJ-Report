* Connect and get to know other members in the AWS First Cloud Journey program.
* Master foundational knowledge of cloud computing and AWS Global Infrastructure.
* Understand how to navigate the AWS Management Console, manage costs, and configure basic security permissions.

### Key Achievements for Week 1:
* Understood what AWS is and mastered the organizational structure of its global infrastructure (Regions/AZs/Edge Locations).
* Successfully initialized and configured an AWS Free Tier account, setting up AWS Budgets to avoid unexpected charges.
* Became familiar with the AWS Management Console interface and learned how to search for and access core services.
* Installed the AWS CLI on the local machine and successfully configured Access Keys/Secret Keys for programmatic infrastructure management.
* Mastered IAM fundamentals: Created and separated permissions for individual users and managed user groups following the principle of least privilege.
