# Summary of "Meetup Session 13-06-2026"

### Purpose of the Event

- Share about Mr. Hieu's journey when participating in the AWS First Cloud AI Journey.
- Share about Tho and Kien's project (URL Shortener project).
- Disseminate knowledge about Data Analytics.
- Disseminate corporate culture.
- What does DevOps actually do? Why choose to be a DevOps Engineer.

- **Danh Hoàng Hiếu Nghị** - AI Engineer – AWS Community Builder – AWS Student Builder Group Leader

### Featured Content

#### Introduction to the First Cloud AI Journey program
- Provided with a clear step-by-step roadmap in the AWS program by Mr. Hieu.
- Pictures shown by Mr. Hieu about the "ranking up" stages in the AWS program.
- And the gifts/swag received when joining Mr. Hieu's team.

#### Introduction to Kien and Tho's project
- URL Shortener project: Presented in great detail by the two members on how it was implemented.
- Discussed the architecture and data flow of the project from user to database.
- The benefits and utility of using the project.

#### Real-world stories to corporate culture at multinational corporations
- Mr. Dat shared what a Data Analytics Engineer actually does in a business, and the essential skills required to become a Data Analytics Engineer.
- Mr. Cuong shared insights about the culture of multinational corporations, ranging from the recruitment process to the long-term challenges of an enterprise.

#### What does DevOps actually do? Why choose to be a DevOps Engineer?
- Discussed the definition of DevOps and the role of a DevOps Engineer.
- How does the DevOps journey look like? What will you learn when choosing DevOps?
- What knowledge to acquire when becoming a DevOps Engineer?
- DevOps tools.
- Mr. Trong introduced the roadmap of what to learn first at the current time.


### Key Takeaways
- A clearer understanding of the Data Analytics industry and corporate culture.
- Knowing how to enter the AWS community as a student or a fresher.
- Knowing how to receive gifts/swag from AWS.
- Understanding the roadmap to learn DevOps, DevOps tools, and what the actual daily tasks of a DevOps Engineer are.
